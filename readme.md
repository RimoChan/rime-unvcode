## 安装⽅法

1. ⾸先你得是rime⽤户……

2. 然后clone这个仓库。

3. 把`opencc`⽂件夹的内容放进rime的程序⽂件夹。

4. 把`unv.yaml`的内容加⼊需要幼⼥code的输⼊⽅案的custom档。

5. 重新部署之后就可以打出幼⼥Code了。
